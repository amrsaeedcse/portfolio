Project Overview

GreenGuardian is a smart irrigation system powered by ESP32 to monitor soil moisture and water levels while automatically controlling the water pump. It aims to optimize water usage and improve plant health.

Key Features

  • Sensors: Soil moisture & Water level detection.
  • Automation: Automatic pump control based on data.
  • IoT Integration: Web/Mobile interface for remote monitoring.
  • Dashboard: Real-time system monitoring.
