Drinks App Still Working on it

Project Overview

This is one of my foundational projects where I started exploring the Flutter framework. It's a simple, lightweight application designed to browse different types of drinks.

The main focus of this project was to master the basics of UI building, Widget composition, and screen navigation without complex backend dependencies.

Key Concepts Learned

  • Flutter Basics: Building layouts using Rows, Columns, and Containers.
  • Navigation: Moving between screens and passing data.
  • UI Design: Creating a clean and responsive interface.
  • Asset Management: Handling images and local fonts.
