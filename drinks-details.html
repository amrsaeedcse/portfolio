Drinks-App Project

Drinks-App Project (Still working on it)

This project is a Flutter-based application for browsing and managing a variety of drinks. It utilizes Firebase as the backend for data storage and user authentication, providing real-time data synchronization and secure access.

  • Firebase Integration: Stores drink data, handles user sign-in/registration, and syncs data between app and cloud.
  • State Management: Uses Cubit for simple states and Bloc for complex, multi-step logic, ensuring clean and scalable code.
  • Flutter UI: Responsive and visually appealing interface for both Android and iOS devices.
  • Status: Final stages of development, focusing on performance, interactive features, and user experience.
