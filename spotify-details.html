Spotify-like Music Streaming App

A full-featured music streaming application built with Flutter, leveraging Cubit and Bloc for robust state management. The app integrates with an external RESTful API for fetching track metadata, playlists, and recommendations, and uses Firebase services for authentication, real-time database operations, and cloud storage.

Key Features

  • User Authentication: Email/Password and Google Sign-In via Firebase Authentication.
  • Music Playback: Smooth streaming with background audio support, play/pause controls, and seek functionality.
  • Dynamic Playlists: Fetch and display curated playlists from the external API.
  • Search Functionality: Real-time search for songs, albums, and artists.
  • Favorites & History: Cloud-synced liked songs and playback history.
  • State Management: Clean and scalable architecture using Cubit and Bloc for predictable state flow.
  • Responsive UI: Adaptive design for mobile and tablet, with smooth animations.
